Shining Star


About

This application visualizes stars from the ATHYG database. Users can travel through space and see various constellations from different cultures. Thie application was developed for CS 528 at University of Illinois at Chicago in Spring 2024. Application was built on Unity 2019.2 for CAVE2.


Features

Menu

Press the menu button on the CAVE2 wand to bring up the main menu of the application. From there, access and apply the various features of the application.

Navigation

The default navigation of this application is freefly. To select a different navigation mode, go to Main Menu -> System -> Navigation Mode. Walk mode is not recommended.

Select Culture for Constellations

The default constellations shown are modern constellations. To view constellations of a different culture, go to Main Menu -> Constellations and select the constellations to view.


To view no constellations and only see the stars, select Main Menu -> Constellations -> No constellations

View Constellation Information

To view information about one of the constellations displayed, go to Main Menu -> Constellations Details.


Constellations must be displayed to see information. If no constellations are displayed, the user will be informed to display constellations in order to display constellation information.

View Exoplanets

To re-color the stars from OBAFGKM to color scheme based on the number of known planets in the system, go to Main Menu -> Exoplanets Mode. The user can choose between OBAFGKM and exoplanet color schemes here.


To see information explaining the color scheme of the exoplanets, go to Main Menu -> Exoplanets Mode -> Exoplanet Color Scheme Information.

View Changes Over Time

To see the stars and constellations move throughout time, go to Main Menu -> Evolution Over Time. The user can choose to go forward or backward in time here. Movement of the stars and constellations can also be stopped within this menu.


The amount of time passed is visible toward center bottom of the screen.

Change Scaling

To change the scaling of distance between the stars, go to Main Menu -> Change scaling.


Default scaling of the application is 1.0x.

Reset Application to Beginning

To reset the application to the location and time when the application first starts, go to Main Menu -> Reset Location and Time.
