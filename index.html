Shining Star

by Meghan Guo


To run this application, download the files from Github

Open the downloaded folder in Unity

Once the files are open in Unity, remove the default scene, drag Assets / shining_star into the hierarchy in the top Omicron menu

Select Configure for CAVE2 Simulator then press play and to have the VR environment running in Unity
