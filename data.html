Shining Star



Star Data

Data Source: ATHYG database Only the "athyg_31_reduced_m10" dataset was used for this application.


PreProcessing:

1. Keep only the following columns: HIP, DIST, X0, Y0, Z0, ABSMAG, MAG, VX, VY, VZ, SPECT

2. Remove all rows without X0, Y0, Z0 data

3. For all remaining rows, update SPECT to keep only the first letter of SPECT value

4. Convert VX, VY, VZ from km per second to parsec per year by dividing existing VX, VY, VZ values by 31.5 (since each meter in CAVE2 maps to 1 parsec)


Constellation Data

Data Source: Stellarium Skycultures database Used each constellationship.fab for the following cultures: modern, hawaiian, navajo, samoan, egyptian.


PreProcessing:

1. Remove all extra space at end of each line if there is one

2. Remove second value of each line

3. Save as text file


Exoplanet Data

Data Source: NASA Exoplanet archive do not download all columns, steps on what to download stated below.


PreProcessing:

1. View and download only HIP ID and Number of Planets columns by selecting them in "Select Columns"

2. Remove duplicate HIP IDs
