Shining Star



Install and Run

Download Files on : Github Only the "athyg_31_reduced_m10" dataset was used for this application.


Build:

1. Download the source code via link above

2. Open the folder in Unity

3. Drag star scene located in Assets and drop it into Hierarchy on the left side of the scene

4. To run the code in Unity only, click the run button and the application should load and begin to play. Make sure the Maximize on Play option is clicked for best experience. STOP HERE if you're running the code in Unity only.

5. To build an executable, go to File->Build Settings and click on Add Open Scenes. shining_star should now appear in the Scenes in the Build list.

6. Click build and select the output folder.

7. Go to the output folder and run shining_star.exe.
